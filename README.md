Tastify is a modern and responsive **food and recipe web application** built with **React**, **React Router**, and **Framer Motion**. Users can browse recipes by category, search dynamically, and view detailed recipe information with smooth page transitions.

## Features

- Browse recipes by categories: Asian, Italian, Indian, Japanese, Mexican, and more.  
- Dynamic search functionality.  
- Detailed recipe pages with ingredients, instructions, servings, prep & cook time, ratings, and tags.  
- Smooth **page transitions** using **Framer Motion**.  
- **LocalStorage caching** for faster loading.  
- Fully **responsive design**.

## Technologies Used

- **React** (Functional Components & Hooks)  
- **React Router v6** (Routing & Dynamic Params)  
- **Framer Motion** (Page transitions & animations)  
- **Axios** (Fetching data from [DummyJSON Recipes API](https://dummyjson.com))  
- **CSS3 / Flexbox / Grid** for layout and styling
